CAN GÖKÇEASLAN

Boğaziçi Üniversitesi Fizik son sınıf öğrencisiyim. Bunun yanında ikinci üniversite kapsamında İstanbul Üniversitesinde Sosyoloji eğitimi almaktayım. İilgi alanlarımdan en önemlisi yazılımdır. HTML / CSS / PHP / JS yapılarını kullanarak websitesi geliştirmekteyim. Bunun yanında source kod yani kaynak kodu geliştirmek üzere C / C ++, Java, Python, Objective-C bilmekteyim. Bunun dışında mobil aplikasyonlar yazmaktayım. Özellikle Android ve IOS tabanlı projelerle uğraşmaktayım.
